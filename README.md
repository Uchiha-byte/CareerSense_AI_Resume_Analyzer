# CareerSense — AI Resume Analyzer

CareerSense is an elite, immersive professional analysis platform. It leverages a tiered AI infrastructure to decode resumes, identify critical skill gaps, and generate cinematic career roadmaps.

## 🚀 Visionary Features

- **Tiered AI Routing**: Intelligence powered by a prioritized model hierarchy (OpenRouter Free → Google Gemini → OpenAI Paid Fallback) for maximum reliability and zero-cost optimization.
- **Exhaustive Data Failover**: A resilient processing loop that automatically switches AI models up to 5 times upon failure, ensuring successful analysis even during high-load periods.
- **Deep ATS Forensics**: Detailed scoring across Keyword Match, Formatting, Readability, and Structure.
- **Recruiter Eye View**: Interactive heatmap showcasing which sections of your profile command professional attention.
- **Gap Analysis & Comparison**: Benchmarking your profile against the top 1% of candidates in your target industry.
- **Ascension Roadmap**: Dynamic 30/60/90-day action plans generated by deep-thinking neural models.

## 🛠️ The Architecture

### Frontend (The UI)
- **React & Vite**: Ultra-responsive interaction layer.
- **Liquid Glass Aesthetic**: A premium design system using layered radial glows, CSS-only hardware-accelerated animations (`will-change`), and semi-transparent backdrop filters.
- **Resilient UI**: Integrated Error Boundaries and cinematic loading state signals.

### Backend (The Core)
- **Node.js & Express**: High-throughput session processing.
- **Intelligent LLM Router**: Priority-based model selection with concurrent session tracking and smart provider penalty logic.
- **Universal Parser**: Hybrid OCR/Text extraction supporting PDF, PNG, and JPEG formats.

## 🏁 Installation Protocol

1. **Environment Initialization**:
   ```bash
   npm install
   ```

2. **Credential Configuration**:
   Create a `.env` in the root (refer to `.env` for the required keys):
   ```env
   OPENROUTER_API_KEY=...
   GEMINI_API_KEY=...
   OPENAI_API_KEY=...
   PORT=3001
   ```

3. **Ignite the System**:
   ```bash
   # Both Backend and Frontend
   npm run dev
   ```
